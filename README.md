# 📥 File Download Simulator
A JavaFX desktop application that simulates downloading multiple files simultaneously using multithreading and thread pools.

## 📌 Project Description
This application demonstrates how multiple file downloads can run concurrently using Java threads.
Each download task runs independently and updates its progress in real time through progress bars displayed in the JavaFX interface.

## ✨ Features
- Simulate multiple file downloads.
- Use multithreading for concurrent execution.
- Real-time progress updates.
- JavaFX graphical user interface.
- Thread pool management using ExecutorService.
- Start multiple downloads simultaneously.

## 🛠 Technologies Used
- Java
- JavaFX
- Multithreading
- ExecutorService
- FXML

## 📂 Project Structure
src/
└── filedownloadsimulator/

## 🚀 How to Run
1. Open the project in NetBeans or VS Code.
2. Build the project.
3. Run `Main.java`.
4. Start download simulations from the interface.
